# Framework #

### Globals  
* 放置一些全局功能的引用

### PanelManager
* 界面管理
* 创建界面
* 显示关闭界面

### LoaderManager
* 资源加载

### ObjectManager
* 管理物件
* 创建物件
* 销毁物件

### Object

### ModeConfig
一个 GameObject 的配置,可通过此配置生产一个 GameObject
